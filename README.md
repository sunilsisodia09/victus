# Victus AI 

Victus AI is a Python-based voice assistant and AI automation system.

It can listen to voice commands, open applications, control browsers, play music, and interact with AI models like Ollama.

---

# Features

- Voice Recognition
- Text To Speech
- Open Chrome
- Open YouTube
- Open Google
- Play Songs
- AI Chat Support
- Offline AI using Ollama
- Browser Automation
- System Automation

---

# Tech Stack

- Python
- Ollama
- SpeechRecognition
- pyttsx3
- pywhatkit
- requests
